# Security Policy

## Supported Versions

| Version | Supported |
|----------|-----------|
| 1.0.x | ✅ |
| < 1.0 | ❌ |

---

## Reporting a Security Issue

If you discover a security issue related to this project, please do **not** publish it publicly.

Instead, report the issue responsibly by contacting the project maintainer.

Please include:

- Description of the issue
- Steps to reproduce
- Expected behavior
- Actual behavior
- Suggested fix (if available)

---

## Responsible Disclosure

This repository is intended for educational and portfolio purposes.

All authentication logs included in this project are **synthetic** and do not contain real users, credentials, or production data.

The project should only be used in authorized environments for learning, research, and demonstration.

---

## Security Best Practices

- Never upload real authentication logs.
- Never include credentials or API keys.
- Never use production security data.
- Follow responsible disclosure practices.
